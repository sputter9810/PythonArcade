# Task 002 - Launcher UI

## Goal

Upgrade the launcher from a placeholder scene into a polished arcade selector.

## Requirements

- keyboard navigation
- mouse hover and click support
- selected card highlight
- footer help text
- title and subtitle
- optional future page support

## UI constraints

- no overlapping text
- card sizing must remain consistent
- launcher must remain readable at base resolution
- use shared theme values
- long titles/descriptions must be safely handled

## Done criteria

- users can select a game card with keyboard and mouse
- the selected state is visually obvious
- the layout remains aligned and readable
