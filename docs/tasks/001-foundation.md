# Task 001 - Foundation

## Goal

Establish a clean, scalable project foundation for the arcade launcher.

## Scope

- app shell
- scene manager
- theme file
- launcher registry
- starter docs
- basic tests

## Done criteria

- project runs
- main menu scene renders
- game select scene renders a 4x4 placeholder grid
- all planned 16 games appear in the registry
- docs explain architecture and UI rules
